This is a project to create a new base theme for Wordpress for the purpose of learning how the wordpress themes work.

#####################################################################################################################

The project is setup to test the process of designing directly from .psd in Photoshop over to Adobe Edge Code and then working with Github as a repository. In this particular project I am also using Dropbox as a backup/additional repository just to see how that works as well.

#####################################################################################################################

While the project is mostly for testing purposes you are more than welcome to join in and add to the code or design.
